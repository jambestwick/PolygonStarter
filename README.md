### 使用方法
### Git下载该工程
安装
Unity 开发工具
https://unity.com/download

导入
File ----> Open Project
选择 Scence 场景
，就进入游戏制作

### 如何将该应用导出使用
File--> Build Setting 设置要编译的平台。
![image](https://user-images.githubusercontent.com/18340126/145790045-418a7d31-361f-4ca2-b050-8f15bd2ac956.png)

Build And run 导出工程直接可以使用。
